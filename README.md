# Семейный бюджет

Небольшой pet-проект: приветственный лендинг + калькулятор семейного бюджета в браузере.

## Что умеет

- Добавление и удаление доходов
- Добавление и удаление расходов
- Планирование будущих платежей с подсветкой просроченных
- Автоматический расчет общего дохода, расходов и баланса
- Круговая диаграмма структуры расходов (Chart.js)
- Сохранение данных и темы в `localStorage`

## Структура

- `index.html` — основной HTML-файл с разметкой лендинга и калькулятора
- `styles.css` — стили интерфейса (тема, таблицы, карточки, адаптивность)
- `script.js` — логика приложения (добавление/удаление записей, графики, localStorage)
- `калькулятор.html` — резервная/предыдущая версия файла
- `LICENSE` — лицензия MIT
- `.gitignore` — базовые исключения для репозитория

## Запуск

1. Скачайте проект.
2. Откройте `index.html` в браузере.

## Короткая инструкция по файлам

- `index.html`: содержит структуру страницы и подключает `styles.css` и `script.js`.
- `styles.css`: отвечает за внешний вид приложения и темную/светлую тему.
- `script.js`: управляет данными бюджета, обновляет таблицы, итоги и графики.
